CBEAM

Defines a beam element.

Format

123456789
CBEAMEIDPIDGAGBX1X2X3OFFT/BIT

Example

CBEAM,9,101,2004,2005,1.0,0.0,0.0

Fields

Field Contents
EID Unique element identification number. (0 < Integer < 100,000,000)
PID Property identification number of PBEAM or PBEAML entry. (Integer > 0)
GA, GB Grid point identification numbers of connection points. (Integer > 0)
X1, X2, X3 Components of orientation vector in the basic coordinate system. (Real)
OFFT/BIT Not used
