CQUAD4

Defines an isoparametric membrane quadrilateral element. In the near future, the entry will be expanded to support isoparametric membrane-bending quadrilateral plate elements.

Format

123456789
CQUAD4EIDPIDG1G2G3G4THETAZOFFS
TFLAGT1T2T3T4

Example

CQUAD4,1,101,4001,4002,3002,3001

Fields

Field Contents
EID Element identification number. (0 < Integer < 100,000,000)
PID Property identification number of a PSHELL entry. (Integer > 0; Default = EID)
Gi Grid point identification numbers of connection points. (Integers > 0, all unique.)
THETA, ZOFFS, TFLAG, T1, T2, T3, T4 Not used.

Remarks:

  1. The continuation is optional.
