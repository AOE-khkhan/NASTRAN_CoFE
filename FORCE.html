FORCE

Defines a static concentrated force at a grid point by specifying a vector.

Format

123456789
FORCESIDGCIDFN1N2N3

Example

FORCE,88,3,,1.0,0.002274,-0.015465,-0.129763

Fields

Field Contents
SID Load set identification number. (Integer > 0)
G Grid point identification number. (Integer > 0)
CID Not used.
F Scale factor. (Real)
Ni Components of a vector measured in default coordinate system. (Real)
