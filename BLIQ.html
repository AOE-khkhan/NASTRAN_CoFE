BLIQ

Canfield bilinear isoparametric quadrilateral membrane element.

Format

123456789
BLIQEIDPIDG1G2G3G4

Example

BLIQ,1,101,4001,4002,3002,3001

Fields

Field Contents
EID Element identification number. (0 < Integer < 100,000,000)
PID Property identification number of a PSHELL entry. (Integer > 0; Default = EID)
Gi Grid point identification numbers of connection points. (Integers > 0, all unique.)

Remarks:

  1. G1-G4 must be in located in a plane parallel to the basic coordinate system x-y plane.
