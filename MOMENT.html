MOMENT

Defines a static concentrated moment at a grid point by specifying a scale factor and a vector that determines the direction.

Format

123456789
MOMENTSIDGCIDMN1N2N3

Example

MOMENT,88,3,,1.0,0.002274,-0.015465,-0.129763

Fields

Field Contents
SID Load set identification number. (Integer > 0)
G Grid point identification number at which the moment is applied. (Integer > 0)
CID Not used.
F Scale factor. (Real)
Ni Components of the moment vector measured in default coordinate system. (Real)
