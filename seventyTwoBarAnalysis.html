Seventy Two Bar Truss Analysis

The seventy two bar space truss was developed for:
Fox, Richard L., and Lucien A. Schmit, JR. "Advances in the Integrated Approach to Structural Synthesis." Journal of Spacecraft and Rockets 3.6 (1966): 858-866.

Example CoFE input files for this page are provided here:
NASTRAN_CoFE/CoFE_examples/a3_seventyTwoBarTrussAnalysis/

MSC Nastran verification files are provided here:
NASTRAN_CoFE/CoFE_examples/a3_seventyTwoBarTrussAnalysis/NASTRAN_comparison/

Linear Statics

Open the seventyTwoBar_analysis.m file and check that the SOL variable is set to 101. Run the script. The postprocessor GUI will show the linear static response compared to the response from MSC Nastran.

  • CoFE element forces can be read from the .c06 output file or FEM.CBEAM.force variable.
  • MSC Nastran element forces can be read from the .f06 output file in the \NASTRAN_comparisons folder.
  • CoFE element stresses can be read from the .c06 output file or FEM.CBEAM.stress variable.
  • MSC Nastran element stresses can be read from the .f06 output file in the \NASTRAN_comparisons folder.

The response has near perfect agreement with MSC Nastran.

Vibration Response

Change the SOL variable to 103 and run the script. The postprocessor GUI will show the first vibration mode shape compared to the result from MSC Nastran.

The response has near perfect agreement with MSC Nastran. The 822 Hz mode is a torsional mode, which is poorly visualized by most postprocessors. Other available results can be selected from the list boxes in the GUI.

Buckling Response

Change the SOL variable to 105 and run the script. The postprocessor GUI will show the buckling mode shapes compared to the results from MSC Nastran.

The postprocessor GUI will initially show the linear static response to the applied reference load. Buckling modes can be selected in GUI list boxes. The response has near perfect agreement with MSC Nastran. Eigenvectors may be scaled differently.
