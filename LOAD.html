LOAD

Defines a static load as a linear combination of load sets defined via FORCE, MOMENT, and GRAV entries.

Format

123456789
LOADSIDSS1L1S2L2S3L3
S4L4-etc.-

Example

LOAD,1025,1.0,1.0,88,1.0,9

Fields

Field Contents
EID Load set identification number. (Integer > 0)
S Overall scale factor. (Real)
Si Scale factor on Li. (Real)
Li Load set identification numbers defined on entry types listed above. (Integer > 0)
